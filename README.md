# Morning Mission App

Static web app ready for GitHub + Vercel deployment.

## Deploy
1. Create a new GitHub repository.
2. Upload `index.html` and `vercel.json` to the repository root.
3. Import the repository into Vercel.
4. Framework preset: `Other`. Build command: none. Output directory: `.`

## Notes
- Camera works best on HTTPS (Vercel deployment).
- If direct camera access is blocked, the app falls back to image upload/capture.
